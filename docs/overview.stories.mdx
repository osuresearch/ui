# Research UI

Research UI is The Ohio State University's Office of Research component library, based on
Ohio State's design system - BUX.

## Installation

Install using your package manager of choice. `@mui/material` and `@emotion/styled` are required peer dependencies.

```sh
yarn add @osuresearch/ui @mui/material @emotion/styled
```

## Usage

Wrap your application in a `<RUIProvider>` to activate themes for child content.

```tsx
import { Typography, Stack } from '@mui/material';
import { RUIProvider } from '@osuresearch/ui';

export function App() {
  return (
    <RUIProvider>
      <main>
        <Typography variant="h1">Hello world!</Typography>
      </main>
    </RUIProvider>
  );
}
```

## Customizing

We support CSS-in-JS through Emotion. See Material UI's guide on styled components:

[https://mui.com/system/styled/](https://mui.com/system/styled/)
