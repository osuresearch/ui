# Typography Tokens

Typography tokens are used to maintain a consistent set of font styles throughout your application.

## Font Family

| Token | Value | Example
| ---- | ---- | -----------
| `--rui-font-sans` | BuckeyeSans, HelveticaNeue, Helvetica, Arial, serif | <Text>The quick brown fox jumped over the lazy dog.</Text>
| `--rui-font-serif` | BuckeyeSerif, Georgia, sans-serif | <Text>The quick brown fox jumped over the lazy dog.</Text>
| `--rui-font-mono` | Menlo, Monaco, 'Courier New', monospace; | <Text>The quick brown fox jumped over the lazy dog.</Text>

## Font Size

Font sizes use rem units so they scale with the base font size.

| Token | Value | Example
| ---- | ---- | -----------
| `--rui-text-xs` | 0.75rem | <Text fs="xs">Aa</Text>
| `--rui-text-sm` | 0.875rem | <Text fs="sm">Aa</Text>
| `--rui-text-base` | 1rem | <Text fs="base">Aa</Text>
| `--rui-text-md` | 1.5rem | <Text fs="md">Aa</Text>
| `--rui-text-lg` | 2rem | <Text fs="lg">Aa</Text>
| `--rui-text-xl` | 2.5rem | <Text fs="xl">Aa</Text>
| `--rui-text-xxl` | 3rem | <Text fs="xxl">Aa</Text>
